# react_todayhome

### react를 활용한 오늘의집 클론코딩

## 💡실행방법

```
git clone https://github.com/mijinyun/react_todayhome.git
npm i
npm start
```

## 📌경로설명

```
- / : 메인
- /store : 스토어 탭 페이지
- /search : 검색결과 페이지
- /users/login : 로그인
- /register : 회원가입
- /experts : 전문가 탭 페이지
- /FAQ : 이사 탭 페이지
```
